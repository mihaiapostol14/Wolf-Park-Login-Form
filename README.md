## Overview

**Wolf Park Login Form** is a simple, visually appealing login form project designed to provide a clean and user-friendly authentication interface. Built primarily using HTML, CSS, and JavaScript, the project demonstrates modern web design practices, with a focus on responsive layout and stylish aesthetics.

- **Author:** [Mihai Apostol](https://github.com/mihaiapostol14)
- **Primary Language:** CSS
- **Visibility:** Public

## Quick Start

### Clone the Repository

```bash
git clone https://github.com/mihaiapostol14/Wolf-Park-Login-Form.git
cd Wolf-Park-Login-Form
```

## Features

- **Modern Login UI:**  
  The form features username and password input fields, a "Remember me" checkbox, and a "Forgot password?" link, all styled for clarity and usability.

  
- **Modern Icons:**  
  The form use icons user,lock, eye from [Font Awesome](https://fontawesome.com/)

- **Responsive Design:**  
  The layout uses flexbox for centering and adapts to various screen sizes.

- **Custom Background:**  
  The background leverages a custom image for visual appeal.

- **Input & Button Styling:**  
  Rounded input fields and buttons with hover effects, as well as icon placeholders for an enhanced user experience.

- **Modular Structure:**  
  - HTML (`html/index.html`): Markup for the login interface.
  - CSS (`css/style.css`): Styles for layout, colors, and effects.
  - Assets (`assets/`): Contains images like the background.
  - (JavaScript file `js/script.js`) show and hide password functionality. 

## File Structure


## How It Works

- Open `html/index.html` in your browser.
- The form appears centered on a blurred background image.
- Enter your username and password, optionally select "Remember me," and submit.
- The design is ready for integration with any backend authentication system.

## Customization

- **Style:**  
  You can change the background image by replacing `assets/image/background.jpg`.
- **Branding:**  
  Modify colors or layout in `css/style.css` to match your branding.
- **Functionality:**  
  Add JavaScript logic in `js/script.js` for form validation or backend connection.

## Usage

Ideal as a starting point for authentication pages in web applications, portfolios, or design demos.
